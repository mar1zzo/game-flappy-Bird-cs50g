## 📑 Descrição do projeto 

🐦 &nbsp; Implementação de níveis mais complicados de jogo onde a velocidade de distribuição e a lacuna entre os tubos sejam aleatórios. Conceder medalhas de gratificação conforme pontuação e acrescentar recurso de pausa.

🐦 &nbsp; Projeto desenvolvido no curso **CS50 Games** oferecido por [Harvard University](https://cs50.harvard.edu/games/2018/weeks/1/).
O CS50 Games é um curso gratuíto online estruturado em 12 semanas oferecido pela Universidade de Harvard com muito conteúdo prático e desafios que levam o participante a aprender gradualmente o desenvolvimento de jogos. 


## 🗃️ Funcionalidades

✔️ &nbsp; Randomização da lacuna entre os tubos (espaço vertical)

✔️ &nbsp; Randomização do intervalo de geração dos pares de tubos na tela

✔️ &nbsp; Implementação de lógica de pontuação mínima e máxima para conceder medalhas ao jogador

✔️ &nbsp; Implementação de recurso de pausa com efeitos sonoros

➕ &nbsp; [Extra] Inclusão de mensagem de tecla de pausa na tela

➕ &nbsp; [Extra] Criação de executável do jogo

1. Löve2D
2. Editor de texto ou uma IDE (ambiente de desenvolvimento integrado), usei o VSCode!

#### 🧭 &nbsp; Rodando o projeto

```bash

$ git clone https://github.com/mar1zzo/game-flappy-bird-cs50g.git

# Existem 2 formas de rodar o jogo:
1) Via executável

# Acesse a pasta do executável do projeto no seu terminal/cmd
$ cd my-flappy-cs50g\exec

# Descompacte o arquivo MyFlappy.zip e terá o executável!
```
